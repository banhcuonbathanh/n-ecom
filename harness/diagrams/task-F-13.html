05

Decisions taken (defaults — say so if you want different)

  • One doc, not four. The reference splits guide/docker/deploy/rollback across files for a multi-person team; our harness is agent-operated — one DEVOPS.md with strong sections beats four files that drift.
  • Ops facts move in now, even though no code exists. Like FE_STATE/BE_STATE, this is a design-ahead doc: F-2/F-4/OPS tasks build against it instead of re-deriving it.
  • Reference gotchas inherited as hard rules, not anecdotes — same treatment F-9 gave the lessons register.
  • Registry = GHCR under the existing repo (ghcr.io/banhcuonbathanh/n-ecom-*) — no new infrastructure decision needed.
