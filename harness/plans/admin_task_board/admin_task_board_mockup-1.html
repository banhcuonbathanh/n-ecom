The table region keeps its shape while loading. In the reference the whole region sits behind a !statsLoading guard, so it unmounts — the page below the KPI row goes blank and then jumps when data lands.

empty-day vs empty-filter — two states, deliberately different
📋
Chưa có công việc nào cho ngày 19/06/2026.
🔍
Không có nhân viên khớp bộ lọc vai trò: chef · trạng thái: hoàn thành.

Same data volume, completely different meaning — one says "nothing was assigned", the other says "your filter hid everything", and each offers the action that actually helps. The reference renders one identical string for both, so a mis-set filter is indistinguishable from an empty day.
