# Hướng dẫn sử dụng: ❤ Yêu thích

Mọi thứ trong tầm tay — xem, điều chỉnh và đặt lại những món bạn thích mà không cần tìm lại từ đầu.

---

## Bước 1: Xem danh sách yêu thích (Zone ZA, ZB, ZC)

| Vùng | Chức năng | Cách dùng |
|------|-----------|-----------|
| ZA — Thanh tiêu đề | Hiển thị tên trang "❤ Yêu thích" và nút giỏ hàng 🛒 | Nhấn ← để quay về thực đơn · Nhấn 🛒 để xem giỏ hàng hiện tại |
| ZB — Bộ lọc loại món | Ba tab: **Tất cả / Món lẻ / Combo** kèm số lượng trong ngoặc | Nhấn tab để lọc — chỉ hiển thị loại bạn chọn |
| ZC — Danh sách món | Mỗi thẻ hiển thị: ảnh · tên · giá · topping đã chọn · tổng/phần | Cuộn để xem tất cả các món yêu thích |

**Điều chỉnh số lượng:**
Nhấn **[−]** để giảm, **[+]** để tăng. Số lượng tối thiểu là 1.
Muốn xoá hẳn khỏi danh sách → nhấn nút ♥ (đỏ) góc phải thẻ.

---

## Bước 2: Thêm tất cả vào giỏ hàng (Zone ZD)

| Vùng | Chức năng | Cách dùng |
|------|-----------|-----------|
| ZD — Footer hành động | Nút chính "Thêm tất cả vào giỏ hàng" + nút lưu set | Nhấn nút cam "🛒 Thêm tất cả vào giỏ hàng" để chuyển toàn bộ danh sách vào giỏ |

Tất cả món trong danh sách yêu thích (theo số lượng đã điều chỉnh) sẽ được thêm cùng một lúc.

---

## Bước 3: Lưu thành set mới (Screen 2 — Zone ZA, ZB, ZC, ZD)

| Vùng | Chức năng | Cách dùng |
|------|-----------|-----------|
| ZA — Tiêu đề | "💾 Lưu thành set mới" + nút quay lại | Nhấn ← để huỷ và quay về |
| ZB — Đặt tên | Ô nhập tên set | Gõ tên dễ nhớ, ví dụ: "Set cuối tuần", "Bữa sáng nhẹ" |
| ZC — Tóm tắt | Danh sách tất cả món kèm topping và tổng tiền | Kiểm tra lại trước khi lưu — chỉ đọc |
| ZD — Xác nhận | Nút "Huỷ" và "💾 Lưu set này" | Nhấn **Lưu set này** để hoàn tất · Nhấn **Huỷ** để quay lại |

Sau khi lưu, bạn được chuyển thẳng đến trang "Các set của tôi".

---

## Bước 4: Dùng lại set đã lưu (Screen 3 — Zone ZA, ZB, ZC)

| Vùng | Chức năng | Cách dùng |
|------|-----------|-----------|
| ZA — Tiêu đề | "📋 Các set của tôi" + nút giỏ hàng | Điều hướng và xem giỏ |
| ZB — Danh sách set | Mỗi set hiển thị tên, số món, tổng tiền và 3 nút hành động | Nhấn **Áp dụng** để thêm tất cả món trong set vào giỏ hàng |
| ZC — Trạng thái rỗng | Hiện khi chưa có set nào | Làm theo gợi ý: về Yêu thích → điều chỉnh số lượng → lưu set |

**Các nút hành động trên mỗi set:**
- **🛒 Áp dụng** — Thêm toàn bộ món của set vào giỏ hàng (cộng thêm, không xoá giỏ cũ)
- **✏** — Đổi tên set
- **🗑** — Xoá set vĩnh viễn

---

## Mẹo & Hỗ trợ đặc biệt

| Tình huống | Cách hệ thống xử lý | Gợi ý cho khách |
|------------|---------------------|-----------------|
| Chưa có món yêu thích nào | Hiện thông báo "Chưa có món yêu thích" | Quay về thực đơn, nhấn ❤ trên món muốn lưu |
| Chưa có set nào | Hiện thông báo "Chưa có set nào" với hướng dẫn | Làm theo hướng dẫn 3 bước bên dưới thông báo |
| Món trong yêu thích đã ngừng phục vụ | Hệ thống tự xoá món đó khỏi danh sách | Không cần làm gì; danh sách tự cập nhật |
| Giỏ hàng đã có sẵn món khi nhấn "Áp dụng" | Cộng thêm số lượng, không xoá món cũ | Kiểm tra giỏ hàng sau khi áp dụng để xác nhận |
| Tắt trình duyệt hoặc đổi thiết bị | Danh sách yêu thích và set được lưu trên thiết bị này | Nếu đổi thiết bị, danh sách sẽ trống trên thiết bị mới |

## Luồng chuẩn

**Tái đặt hàng nhanh:**
```
Vào ❤ Yêu thích
  → Kiểm tra số lượng [−/+]
  → Nhấn 🛒 Thêm tất cả vào giỏ hàng
  → Tiến hành thanh toán
```

**Tạo và dùng set:**
```
Vào ❤ Yêu thích
  → Điều chỉnh số lượng
  → Nhấn 💾 Lưu thành set mới
  → Đặt tên → Lưu set này
  → (Lần sau) Vào 📋 Các set → Áp dụng
```
